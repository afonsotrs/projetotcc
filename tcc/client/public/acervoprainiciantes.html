Este site é um projeto desenvolvido por alunos do IFRN que servirá de material para a produção de um TCC. Nele, o usuário poderá receber recomendações de livros baseado em suas preferências que serão capturadas por um formulário.

O site possui 3 formulários para 3 acervos difentes de livros, além disso, possui um sistema de login no qual recebe o email e senha do usuário para realizar o cadastro
